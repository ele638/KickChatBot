Required:
1) Installed PostgreSQL server
2) ENV variables:
* DB_URL - database URL (ex: localhost)
* DB_PORT - database port (ex: 5432)
* DB_NAME - name of database (lowercase only)
* DB_USER - database username (lowercase only)
* DB_PASSWORD - database password
